
# LiveSearch

Livesearch desenvolvido com perl e o microframework Dancer.
Para instalar o ambiente e startar a aplicação, faça:

  cmod 700 install_environment.sh
  ./install_environment.sh
  
Para somente startar a aplicação:
  ./bin/app.pl
  
Todos os retornos são baseados no json contido no arquivo gistfile1.json.

Existem melhorias a ser feitas, porém o objetivo foi tornar a leitura do código fácil e não dependente de conhecimentos 
prévios da linguagem/framework escolhidos para a criação da aplicação.
